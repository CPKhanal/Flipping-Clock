# 🕰️ Modern Flip Clock PWA

A sleek, minimalist digital flip clock with smooth 3D animations. Designed to be a professional desk companion or a full-screen aesthetic screensaver.



## 🚀 Live Demo
**[flippingclocks.netlify.app](https://flippingclocks.netlify.app/)**

## ✨ Features
* **3D Flip Animation:** Realistic mechanical flip effect for hours, minutes, and seconds.
* **PWA Ready:** Installable on Desktop, iOS, and Android for a native app experience.
* **Offline Support:** Fully functional without an internet connection once cached.
* **Adaptive UI:** Fluid typography and scaling that looks great on mobile, tablet, and ultra-wide monitors.
* **Format Toggle:** Switch between **12-hour** and **24-hour** formats instantly.
* **Immersive Mode:** Supports native fullscreen for a distraction-free experience.

## 🛠️ Built With
* **HTML5 & CSS3:** Utilizes CSS variables and Flexbox for high-performance rendering.
* **Vanilla JavaScript:** Lightweight logic with zero external dependencies or frameworks.
* **Service Workers:** Handles asset caching for offline reliability.
* **Web App Manifest:** Configured with maskable icons for a polished OS integration.

## 📲 Installation
1. Open [flippingclocks.netlify.app](https://flippingclocks.netlify.app/) in your browser.
2. **Desktop (Chrome/Edge):** Click the **Install icon** in the address bar.
3. **Mobile (Android):** Tap the **"Add to Home Screen"** banner.
4. **Mobile (iOS):** Tap **Share** → **Add to Home Screen**.
